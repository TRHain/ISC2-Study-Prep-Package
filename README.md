## How to Use

1. Start with the **[ISC2-CC-Study-Guide.md](ISC2-CC-Study-Guide.md)** file for a condensed review of all 5 CC domains.
2. Use the Rapid-Fire Self-Test at the end of the guide to quiz yourself.
3. Focus on areas where you feel weakest.
4. Memorize the tables (ports, risk treatments, access control models, IR phases).
5. Review the ISC2 AI Exam Guidance section for any AI-related questions.
